# 🎵 Spotify Frontend Clone

A Spotify-inspired frontend web UI built using **HTML and CSS only**.  
This project focuses on layout design, responsiveness, and UI recreation skills.

---

## 🚀 Features

- 🎧 Spotify-like modern UI design
- 📱 Responsive layout for different screen sizes
- 🎨 Clean sidebar + main content structure
- 🖼️ Playlist and album-style UI cards
- 💡 Pure HTML & CSS (no JavaScript)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
